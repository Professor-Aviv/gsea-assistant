Explore GSEA results

This tab merges two files produced by GSEA into one unified view. For each enriched pathway (from the enrichment report), it finds the matching column in the leading edge matrix and appends the leading edge genes — the specific genes that drove the enrichment score. The result is a single table where each pathway row also contains its gene list, which you can explore interactively or download as Excel.
